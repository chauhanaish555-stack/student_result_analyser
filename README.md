## Student Result Analyser

A data analysis project built with Python to analyse student marks.

## What it does
- Calculates average marks for each student
- Finds the topper
- Identifies students who need attention
- Plots a bar graph of results

## Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib

## How to Run
- Install Anaconda
- Open the .ipynb file in Jupyter Notebook
- Run each cell one by one
